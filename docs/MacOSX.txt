To intall on MacOSX:

* Install XCode
* Install homebrew
 - https://github.com/mxcl/homebrew/wiki/installation
* easy_install virtualenv
* mkdir ~/virtenvs && cd ~/virtenvs
* virtualenv --no-site-packages fordrop
* . ~/virtenvs/fordrop/bin/activate
* Download Jpype (needed for neo4j-embedded)
* Do the following:
 - http://www.djangoshmango.com/?p=126
* python setup.py install
* Install libjpeg (needed for jpeg support in python-PIL):
 - http://stackoverflow.com/questions/8404956/installing-pil-with-jpeg-support-on-mac-os-x
* brew install libmagic (needed for python-magic)
* pip install -r requirements/fordrop.pip
